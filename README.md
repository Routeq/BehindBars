# Behind Bars - Premium Bicycle Shop

A modern, dark-themed website for Behind Bars bicycle shop, featuring an interactive forum and comprehensive product catalog.

## Features

- **Modern Dark Theme**: Inspired by contemporary forum design with gradient accents
- **Responsive Design**: Fully responsive layout that works on all devices
- **Interactive Navigation**: Smooth page transitions between sections
- **Product Catalog**: Browse bikes by category (Road, Mountain, Hybrid, E-Bikes, Accessories)
- **Services Section**: Complete list of repair and maintenance services
- **Community Forum**: Interactive forum for cyclists to share tips, routes, and discussions
- **Contact Form**: Easy way for customers to get in touch

## Sections

1. **Home**: Welcome section with key features
2. **Shop**: Product catalog with filtering
3. **Services**: Available bike services and pricing
4. **Forum**: Community discussion board
5. **About**: Company information and statistics
6. **Contact**: Contact information and form

## Technologies Used

- HTML5
- CSS3 (with CSS Variables and Grid/Flexbox)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts (Rajdhani)

## Design Theme

The website features:
- Dark background (#0a0a0a)
- Crimson accent color (#dc143c)
- Orange secondary accent (#ff6b35)
- Glass-morphism effects
- Smooth animations and transitions
- Modern card-based layout

## Getting Started

Simply open `index.html` in a modern web browser to view the website.

## Customization

To customize the website:
- Edit colors in `styles.css` (CSS variables at the top)
- Modify products in `script.js` (products array)
- Update forum posts in `script.js` (forumPosts array)
- Replace logo files (`logo.png` and `logo.svg`)
